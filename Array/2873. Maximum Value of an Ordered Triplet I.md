# 2873. Maximum Value of an Ordered Triplet I

## 1. **從頭推到尾的貪心策略**
- 採用 **Greedy** 方法，從陣列開頭開始遍歷，逐步推導最佳解。
- 每次遇到新數字時，根據策略進行選擇，以確保局部最優解能導向全域最優解。

---

## 2. **將每個數字視為三種角色**
- 每個數 `nums[i]` 都可以被視為 **三種不同的數值**：
  - `nums[k]`
  - `nums[j]`
  - `nums[i]`

---

## 3. **分別尋找最大值**
- 在遍歷過程中，動態維護這三種數值。
- 在每次更新時，就能確保 `i ≤ j ≤ k`。
- 保持 **貪心選擇**，在當前可行範圍內找到最大的數。
